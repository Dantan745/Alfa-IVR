**[ ОБРАЗ ПРОДУКТА ]**

Полноценное мобильное приложение для Android "Alfa Кошелек ЦР", которое будет размещено на RuStore. С моей стороны (дизайн мобильного приложения в Figma, разработка на Android и работа с PostgreSQL) реализую следующие функциональные требования:

1. Безопасная аутентификация: вход с двухфакторной проверкой и управление сессиями

2. Главный экран: отображение баланса, последних операций и быстрый доступ к функциям

3. Мгновенные P2P-переводы: переводы по номеру телефона или ID

4. История операций: детализированный список с фильтрацией по дате, типу и контрагенту

5. Офлайн-режим: просмотр истории и подготовка платежей без интернета

6. Управление лимитами: настройка дневных/месячных лимитов с уведомлениями

7. Адресная книга: сохранение и управление списком частых контрагентов

8. QR-коды: генерация и сканирование для быстрых платежей

9. Массовые выплаты: загрузка CSV-файла для отправки множества платежей одной операцией

10. Визуальный конструктор смарт-контрактов: настройка автоматических платежей

11. Интеграция с ML-моделями: отображение уведомлений об аномальных операциях и прогнозах платежей

Архитектура данных: серверная PostgreSQL для основного хранения, Retrofit для API-взаимодействия с бэкендом. Технологии: Kotlin, Android Studio, Jetpack Compose, PostgreSQL, дизайн на Figma для мобильного приложения.
